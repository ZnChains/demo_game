# 贪吃蛇游戏
#仅仅为了测试而产生的简单demo
这是一个使用纯前端技术（HTML、CSS、JavaScript）开发的贪吃蛇游戏。游戏具有精美的界面、穿墙功能和老鼠造型的食物。

## 游戏特性

- 🎮 **直观的控制**：支持键盘方向键、WASD键控制，以及触摸屏滑动控制
- 🖼️ **精美的界面**：采用现代化UI设计，包括渐变背景、卡片式布局和响应式设计
- 🐍 **穿墙功能**：贪吃蛇撞到墙壁后会从另一面穿出，增加游戏趣味性
- 🐭 **老鼠造型食物**：食物以可爱的老鼠形象显示
- 📈 **等级系统**：随着分数增加，等级提升，游戏速度逐渐加快
- ⏸️ **游戏控制**：支持暂停/继续、重置游戏功能

## 如何运行游戏

1. 找到游戏文件夹 `d:/trae_workspace/demo_game`
2. 双击 `index.html` 文件，游戏将在默认浏览器中打开
3. 点击"开始游戏"按钮开始玩游戏

## 游戏控制

- **方向键** 或 **WASD键**：控制贪吃蛇的移动方向
- **空格键**：暂停/继续游戏
- **回车键**：开始/重置游戏
- **触摸屏设备**：在屏幕上滑动控制方向

## 游戏规则

- 贪吃蛇吃到老鼠会增加分数并变长
- 撞到自己的身体会导致游戏结束
- 蛇可以穿墙而过，从另一面墙出来
- 随着等级提升，游戏速度会逐渐加快
- 得分越高，等级越高

## 开发说明

游戏使用以下技术开发：
- HTML5 Canvas：绘制游戏画面
- CSS3：实现游戏界面样式
- JavaScript：游戏逻辑控制

文件结构：
- `index.html`：游戏HTML结构
- `style.css`：游戏样式文件
- `script.js`：游戏逻辑实现
- `.gitignore`：Git忽略文件配置

## 注意事项

- 确保您的浏览器支持HTML5和Canvas
- 游戏支持PC和移动设备，在移动设备上使用触摸控制
- 如果游戏画面显示异常，请刷新页面重试
